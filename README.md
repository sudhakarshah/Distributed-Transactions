# Distributed-Transactions

Make : ```make```<br />
Start deadlock detector : ```python3 deadlock.py 9999```<br />
Start 5 servers : ```python3 run.py```<br />
Run client : ```./client num(0-9)```<br />

## Example <br />
```./client 0```<br />
```./client 1```<br />

## Features <br />
Atomic, Consistency, Isolation with deadlock detection <br/>

<br />
Made using Go
